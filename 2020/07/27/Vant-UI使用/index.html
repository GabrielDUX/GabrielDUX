Vant的使用

导入项目

执行以下步骤：

  1. 找到小程序的根目录（就是工程文件夹）

  2. cd到这个根目录，或者直接在根目录里git bush

  3. 执行npm init，一直回车，执行默认设置即可

  4. 执行npm i vant-weapp -S --production

  5. 回到开发者工具，点击工具 -> 构建npm。这一步如果弹出本地设置，则勾选使用npm模块。没有则打开工具 -> 项目详情->本地设置，勾选使用npm模块

遇到的问题

在使用过程中发现有些组件npm包中是没有的。因为有赞没有给npm包更新。解决方法是直接下载源码：https://github.com/youzan/vant-weapp/tree/dev/dist

然后将dist直接复制到项目中，然后直接使用即可。
