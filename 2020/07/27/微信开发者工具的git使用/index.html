微信开发者工具的git使用
发表于 2020-07-27| 更新于 2020-07-27
阅读量:

1. 推送代码

我在本地进行了README.md的修改：

在版本管理中可以看到：

勾选这个README，然后再在下面输入标题即可提交。本地进行的工作会保存在工作区，选取自己需要的进行提交即可。提交后，在本地分支的master即可看到自己的提交。选中需要提交的版本，点击“推送”，选择需要的选项，即可进行推送。

打开微信开发者·代码管理，即可看到对应的分支文件被更新：

对应输入的提交标题即为“最后的提交”的信息。

2. 下载代码

使用“抓取”，从远程仓库抓取代码，得到远程仓库中的代码。要将本地代码改为远程仓库中的代码，可以将HEAD重置到那个版本的代码（HEAD指当前版本的代码）。如：

这里选中：将HEAD重置到"a5da5f"。这串字符指代码版本的编号，可以在每个版本的左下角看到。

这里取消勾选保持工作区的内容，然后确定。修改过后，可以看到本地分支变成了test分支，而HEAD变成了a5da5f。同时，返回查看代码，发现代码被更新。
